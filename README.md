# E-Commerce Static Template

Una solución elegante y completa para tiendas en línea estáticas.  
Este proyecto ofrece una base sólida de **UI/UX moderna**, optimizada para dispositivos móviles y lista para integrarse con cualquier backend o plataforma de e-commerce.

---

## Características

- **Ecosistema completo de páginas**  
  Incluye catálogo, carrito, checkout y blog.

- **Diseño responsive**  
  Adaptado a móviles, tablets y escritorio usando **Bootstrap**.

- **Componentes UI premium**  
  Sliders dinámicos, selectores personalizados y galerías interactivas.

- **Optimización visual**  
  Transiciones suaves, efectos hover y carga eficiente de imágenes.

- **Arquitectura escalable**  
  Uso de **SCSS** para una mejor organización y personalización de estilos.

---

## Estructura del Proyecto

```
├── css/                 # Estilos compilados y librerías CSS
├── js/                  # Scripts y lógica de interacción
├── scss/                # Archivos fuente de Sass
├── vendors/             # Plugins de terceros (Owl, NiceSelect, etc.)
├── img/                 # Recursos visuales
│
├── index.html           # Página principal
├── category.html        # Catálogo de productos
├── single-product.html  # Detalle de producto
├── cart.html            # Carrito de compras
├── checkout.html        # Proceso de pago (demo)
├── blog.html            # Blog / noticias
└── contact.html         # Formulario de contacto
```

## ⚙️ Instalación y Uso

### 📋 Requisitos

- Navegador web moderno
- (Opcional) Herramientas de desarrollo:
  - VS Code + extensión Live Server
  - Live Sass Compiler o Node-Sass

### 🔧 Pasos

1. **Clonar el repositorio**

```bash
git clone https://github.com/tu-usuario/nombre-del-repo.git
```

2. **Abrir el proyecto**  
   Abre `index.html` directamente en tu navegador, o usa Live Server para desarrollo en tiempo real.

3. **Procesamiento de formularios**  
   `contact_process.php` requiere un servidor local: XAMPP, Laragon o WAMP.

Puedes adaptar fácilmente la plantilla a tu marca:

- 🎯 **Colores y tipografía** — Modifica las variables en `/scss`
